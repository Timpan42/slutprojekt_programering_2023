Ett multiplayer tank spel

Viktigt att köra i konsolen/cmd!

Kommando: java -jar [directory]\server.jar

Kommando: java -jar [directory]\client.jar

Server

Klient

När man trycker på Ready knappen skickas information till servern från klienten. Om man väljer Heal, Move eller Attack före man trycker Ready skickas speciell information till servern som är relaterade till dem tre valen.

Dem gör inget just nu förutom att skicka informationen till servern. Spel brädan funkar inte just nu, den uppdaterar inte.

Har också en bug där om man börja med att försöka röra på sig innan man gör något annan så kan allt krascha
